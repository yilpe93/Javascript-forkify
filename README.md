## Forkify

Modern Javascript: ES6, NPM, Webpack, Babel, API, MVC pattern 학습

[http://food2fork.com/](http://food2fork.com/)의 API를 활용한 Recipes Project

### Getting Started

#### Installing

```bash
$ npm install
```

#### Running
```bash
$ npm run start
```

검색은 `pasta` 또는 `pizza`로 하시면 빠르게 테스트하실 수 있습니다.
